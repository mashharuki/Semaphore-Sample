# Product Overview

Semaphore プロトコルを活用した、匿名フィードバック投稿システムとアカウントアブストラクション（AA）のデモアプリケーション群。

## Core Capabilities

- **匿名フィードバック送信**: Semaphore ゼロ知識証明を使用した、身元を明かさないフィードバック投稿
- **グループベース匿名性**: メルクルツリーによるメンバーシップ証明と二重送信防止
- **アカウントアブストラクション統合**: Biconomy などの AA スタックを使ったガスレストランザクション
- **マルチネットワーク対応**: Sepolia、Base Sepolia など複数のテストネットワークでのデプロイ

## Target Use Cases

- Semaphore プロトコルの基本的なユースケースのデモンストレーション
- ゼロ知識証明を活用した匿名投票・フィードバックシステムのリファレンス実装
- アカウントアブストラクションの実装例と学習教材
- Web3 における匿名性とプライバシー保護の実験的プロトタイプ

## Value Proposition

ゼロ知識証明技術と最新の Ethereum スタック（Account Abstraction、ERC-7579）を組み合わせることで、ユーザーのプライバシーを保護しながら、検証可能な匿名アクションを実現する。開発者向けには、モノレポ構成で複数のアプローチを比較検討できる教育的価値を提供。
